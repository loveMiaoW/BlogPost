##### scanf/printf

printf(const char* format,...)

format: "%[修饰符]格式字符"

```c++
d i //有符号十进制
x X //十六进制无符号
o	//八进制无符号
u	//无符号十进制
c	//单一字符
s	//字符串
e E //指数浮点小数
f 	//小数浮点数
%% 	//百分号
```

```c++
//修饰符
cm 宽度 左补c
- 左对齐 默认右对齐
+ 正数显示+
0 左侧补零
# 加上 0x/0
l + d/f/x/u double 类型数值
在数据后边加上单位LL肯定不会溢出
gcc -Wall 输出警告
```
