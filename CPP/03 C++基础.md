### 基本语法

c++程序可以被定义为一个对象集合，彼此间通过调用对方的方法进行通信。

- 对象:对象具有行为和状态。例如:一只小狗拥有状态-颜色，名称，品种以及行为-摇尾巴，吃。对象是类的一个实例。
- 类:一个类可以定义为一个模板，它描述的是它所支持类型对象的行为或者状态。
- 方法:基本上指一个行为。一个类可以包含多个方法。
- 实例变量:每个对象有独特的实例变量集合。

#### 函数

函数是执行某个任务的一组语句。

每个c++程序必须至少有一个功能,`main()`。

定义一个函数

```c++
return_type function_name (paramter list)
{
	function body
	return some_thing
}
```

包含一下几部分

- 返回值类型
- 函数名称
- 参数
- 函数体

```c++
//max函数，返回俩个数的最大值
int max(int num1,int num2)
{
    if(num1 > num2)
    {
        return num1;
    }
    else
    {
        return num2;
    }
}
```

##### 函数声明

一个函数声明告诉编译器函数的名字和如何调用这个函数。函数的实际实体可以单独定义。

函数声明有一下几部分

```c++
return_type function_name (parameter list);
int max(int num1,int num2);//max函数声明。参数的名字并不重要，但是类型必须正确，所以这样也是可以的
int max(int,int);
```

当我们在一个源文件中定义了一个函数，并且要在另一个文件调用该函数，函数声明是必须的。

应该在调用该函数的文件的顶部声明这个函数。

##### 函数参数

如果一个函数要是有参数，它必须声明接受参数值的变量。这些变量称为函数`形参`。

想其他本地变量一样，这些`形参`在函数中，在进入函数时创建，在函数退出时销毁。

函数参数传递方式如下

| 调用类型 | 描述                                                         |
| -------- | ------------------------------------------------------------ |
| 值调用   | 将参数的实际值复制到该函数的形参。此情况下，在函数中对该参数做出的任何更改对这个参数没有影响。 |
| 指针调用 | 将参数的地址复制到形参。在函数内部，这个地址用来在调用中访问这个实参。这意味着，对参数所做出的更改会影响参数。 |
| 引用调用 | 将参数的引用复制到形参。在函数内部，这个引用用来在调用中访问这个形参。这意味着，都参数的更改会影响参数。 |

##### 参数默认值

当定义一个函数，可以为最后一个或者几个指定一个默认值。当调用函数时候，如果对应参数为空，将使用此值。

```c++
int sum(int a,int b，int c);
int sum(int a,int b,int c = 10);		//可以
int sum(int a,int b = 10,int c);		//不可以
```
