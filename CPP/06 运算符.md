### 运算符

#### 算数运算符

++ / --

i++ -> 表达式 i i为2

++i -> 表达式2 i为2

#### 关系运算符

得出的都是关系运算符

= 赋值

== 判断是否相等

逻辑运算符短路特性 && ||  (前边为假 后边条件不执行)

扩展运算符

+= -= /= 

```c++
add += 10; add = add + 10;
```

```c++
add = 6; // 9 81 0
add -= a *= a += 3;// 0 变量会改变 后边的程序可能会改变前边的值
```

#### 三目运算符

op1 ？ op2 : op3

```c++
a > b ? a : b
```

#### 逗号

一般用在循环条件，并列指定变量

```c++
sizeof() //变量在目前环境所占的字节数
```

#### 类型转化

强转，不改变原值 -> 拷贝函数
